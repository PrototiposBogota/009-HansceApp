### Ejecución básica

Los servicios alojados en el backend de AnzenApp ya se encuentran publicados a través de Heroku, por lo cual no se requiere nada para realizar la ejecución, solo basta con utilizar la App para que realice el consumo de dichos servicios.

El backend cuenta con dos entidades:

ReportEntity
UserEntity

Las cuales se encargan de realizar todas las operaciones en la base de datos de acuerdo con los servicios expuestos para tal fin.

por ejemplo para obtener los reportes realizados la ruta sería:

https://security-server-inpahu.herokuapp.com/reports
